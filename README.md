# Pro/g/ramming Challenges v5.0

> *"What I cannot create, I do not understand." — Richard Feynman*

A curated list of **250 programming challenges** spanning 10 domains, from beginner to expert. This is the fifth edition — a superset of the classic v4.0 with 105 new challenges drawn from modern software engineering, systems programming, AI/ML, and the legendary [build-your-own-x](https://github.com/codecrafters-io/build-your-own-x) repository.

## How to Use

1. Pick a challenge that interests you
2. Build it from scratch in the language of your choice
3. Push your code to a public repository
4. Repeat

**Difficulty Levels:**
- 🟢 **Easy** — Beginner-friendly, basic concepts
- 🔵 **Medium** — Moderate complexity, multiple components
- 🟠 **Hard** — Complex architecture, systems-level thinking
- 🔴 **Expert** — Professional-grade, deep knowledge required

Every challenge designated `[v4]` is carried over from Programming Challenges v4.0.
